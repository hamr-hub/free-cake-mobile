# 业务规则

## 一、活动规则
- 每个村镇赛区两周一期
- 每期每个赛区最多产生 100 名免费获奖用户
- 免费奖品固定为 6 寸动物奶油基础款
- 用户仅可从系统生成的标准化设计中选择，不支持获奖后再改稿

## 二、参与规则
- 用户须完成手机号验证
- 用户须选择所属赛区
- 用户须通过定位 / IP / 历史行为综合判断在 10km 服务范围内
- 每个手机号每期仅可参赛 1 次

## 三、投票规则
- 每个账户每天最多投 3 票
- 单设备 / 单手机号 / 单 OpenID / 单账号对同一作品的投票频率受限
- 命中风险策略的票数先冻结后复核
- 排行榜展示实时票数与有效票数分离

## 四、获奖规则
- 活动截止后按有效票排序
- 若有效票相同，则按发布时间优先，其次按风控分更低者优先人工复核
- 平台保留对刷票票数扣减和取消资格的权利

## 五、生产规则
- 仅下发标准化可生产款式
- 单节点设置日产能上限
- 超出日产能的订单自动延后排期或切换人工兜底流程

## 六、核销规则
- 获奖用户须本人到店
- 核销方式：手机号 + 核销码 + 门店扫码确认
- 每个核销码仅可成功使用一次
- 超过领取时效自动失效

## 七、库存规则
- 原料由云端统一采购与下发
- 节点仅可申领，不可自由替换核心原料标准
- 库存低于安全阈值自动预警

## 八、申诉规则
- 用户可对被扣减票数、未获奖结果、核销失败发起申诉
- 平台需保留投票日志、风控日志、核销日志用于审计
