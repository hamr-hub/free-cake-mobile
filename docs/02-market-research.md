# 市场调研

## 一、调研结论
结合公开资料与行业共识，可形成以下判断：

1. 烘焙消费持续向低线城市、县域、乡镇扩散，下沉市场仍存在增长空间。
2. 乡镇用户对生日、节庆、婚庆、儿童相关蛋糕具备稳定刚需，但对价格、交付确定性、口碑更敏感。
3. 下沉市场更适合“高性价比、强标准化、低租金、小团队”的门店模型，而非重装修、重堂食模型。
4. 免费活动类产品如果缺乏区域限制和防刷机制，极易被羊毛党、刷票团队、跨区用户利用。
5. 3D 食品打印 / 自动化生产具备展示价值与标准化潜力，但在真实商业场景中仍需面对原料稳定性、设备维护、食品安全、异常兜底等问题。

## 二、对项目有价值的行业规律
### 1. 下沉消费规律
- 用户对“免费 + 熟人传播 + 到店领取”敏感，适合裂变获客。
- 用户更看重拿到实物、口味稳定、取货方便，而不是复杂定制。
- 门店只要能覆盖核心镇区与周边 10km，即可形成局部密度优势。

### 2. 蛋糕业务规律
- 生日蛋糕属于低频但高确定性需求。
- 标准款蛋糕更适合成本控制和批量生产。
- 节假日、周末、寒暑假、婚庆旺季会形成明显峰值。

### 3. 活动运营规律
- 免费奖品要足够吸引，但成本必须锁死。
- 用户参与路径必须足够短，否则分享率和转化率下降。
- 排行榜类玩法必须可审计，否则容易引发信任危机。

## 三、竞品 / 替代方案观察
### 1. 传统蛋糕店
优势：履约成熟、口味稳定。  
劣势：获客效率低、数字化弱、很难做跨镇统一运营。

### 2. 连锁线上蛋糕品牌
优势：线上下单、供应链成熟。  
劣势：深入乡镇末端履约时成本较高，难以做“单镇社群运营 + 自提活动”。

### 3. 普通投票活动平台
优势：搭建快。  
劣势：缺乏与履约、生产、区域限制联动，容易沦为纯拉票工具。

## 四、项目机会点
- 用免费活动快速获取本地用户池
- 用统一中台把多镇运营标准化
- 用标准化蛋糕降低履约成本
- 用社群和未获奖转付费提升 LTV

## 五、项目约束
- 食品行业必须优先保证安全和合规
- 首期不应把商业模型完全建立在复杂机器人生产上
- 区域风控、投票风控、核销风控是成败关键基础设施
